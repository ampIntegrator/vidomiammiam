# Upto1

Expert en croissance digitale - Website built with Payload CMS and Next.js.

## Overview

This is the official Upto1 website featuring:

- **Custom Blocks**: HeroSection, Statistics, IconsBox, FAQ, Testimonials, ClientSlider, ColoredBanner, NumberedFeatures
- **Multi-step Forms**: 5-step SEO audit form with validation
- **SEO Optimized**: Schema markup, meta tags, and structured data
- **Industry Pages**: E-commerce, TPE & PME, PrestaShop solutions
- **Service Pages**: SEO, Growth Hacking, Digital Strategy

## Quick Start

```bash
# Install dependencies
pnpm install

# Copy environment variables
cp .env.example .env

# Start development server
pnpm dev
```

Visit `http://localhost:3000` to view the site.

## Tech Stack

- **CMS**: Payload CMS v3.63.0
- **Framework**: Next.js 15.4.7
- **Database**: LibSQL/SQLite
- **Styling**: TailwindCSS + Custom CSS
- **Icons**: Nucleo Icons
- **Fonts**: DIN Alternate Bold

## Development

The project uses:
- TypeScript for type safety
- Payload blocks for content management
- Server-side rendering with Next.js App Router
- SQLite for local development

## Database Seeding

Access the seed endpoint at `/next/seed` or use the admin panel to populate the database with sample content.

Access the admin panel at `/admin` with your credentials.
